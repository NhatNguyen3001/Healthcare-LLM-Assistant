MASS-RAG · Multi-Agent Source-Specific Retrieval

query → ChromaDB top-k → per-source filter agents (parallel) → synthesis → final prompt context

Retrieval

  • • ChromaDB cosine, single collection
  • • namespaced ids: source_origId
  • • k=7 final, over-fetch k×3 then dedupe
  • • threshold 0.8 → 0 survivors fires web_search fallback

Filter agents

  • • One per source present in top-k (max 6)
  • • Each: GPT-5.4-mini, source-aware prompt
  • • Sees only its own source's chunks
  • • Parallel ThreadPoolExecutor, no shared state

Synthesis & output

  • • GPT-5.4-mini merges per-source outputs
  • • Deduplicates by normalised text fingerprint
  • • Ranks + condenses to a compact context
  • • Concatenated with STO/NER for final prompt
