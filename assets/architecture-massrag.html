MASS-RAG · Multi-Agent Source-Specific Retrieval

query → ChromaDB top-k → three filter agents (Summarizer · Extractor · Reasoner) in parallel → synthesis → final prompt context

Retrieval

  • • ChromaDB cosine, single collection
  • • namespaced ids: source_origId
  • • k=7 final, over-fetch k×3 then dedupe
  • • threshold 0.8 → 0 survivors fires web_search fallback

Filter agents (3 roles)

  • • Summarizer: abstractive, query-relevant summary
  • • Extractor: verbatim spans, tagged with [doc_id]
  • • Reasoner: cross-doc agreement / contradiction
  • • All GPT-5.4-mini, ReAct scaffold, see same retained hits
  • • Parallel ThreadPoolExecutor (max_workers=3)

Synthesis & output

  • • GPT-5.5 reconciles the 3 agent outputs
  • • Resolves contradictions explicitly
  • • Preserves Extractor's [doc_id] citations
  • • Evidence only, no recommendation
  • • Concatenated with STO / NER for final prompt
