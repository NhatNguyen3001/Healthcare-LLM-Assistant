Healthcare NLP Assistant · System Architecture

voice / text → privacy filter → multi-agent RAG → sidebar-selectable LLM → clinical recommendation

Privacy first

  • • Railguard masks PII before any LLM call
  • • regex (phones, emails, addresses, dates)
  • • scispaCy NER (person names, locations)
  • • age bands instead of literal ages

Two-LLM design

  • • Filter agents always cloud (GPT-5.4-mini)
  • • Final generator: GPT-5.5 OR QLoRA Qwen
  • • Sidebar switch is live, per-turn
  • • Both pass through MASS-RAG identically

Knowledge base

  • • 6 sources, ~56k cleaned chunks
  • • S-PubMedBert embeddings
  • • ChromaDB cosine, k=7, threshold 0.8
  • • Dedupe + per-source dispatch
